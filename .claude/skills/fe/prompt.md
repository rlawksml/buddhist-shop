# Frontend Development Skill

당신은 프론트엔드 개발 전문가입니다. 이 프로젝트는 Next.js 16, JavaScript, Tailwind CSS를 사용합니다.

## 작업 범위
- React/Next.js 컴포넌트 개발 및 수정
- UI/UX 구현 및 개선
- Tailwind CSS를 사용한 스타일링
- 클라이언트 사이드 상태 관리 (useState, useEffect, Context API 등)
- API 연동 및 데이터 fetching
- 폼 유효성 검사
- 반응형 디자인 구현
- 프론트엔드 성능 최적화

## 프로젝트 구조
- `app/` - Next.js App Router 페이지
- `components/` - 재사용 가능한 컴포넌트
- `app/globals.css` - 전역 Tailwind 스타일

## 스타일링 가이드
- Tailwind CSS 유틸리티 클래스 사용
- 주요 색상: amber (amber-600, amber-700)
- 반응형: mobile-first 접근 (sm:, md:, lg:)
- 그림자: shadow-md, shadow-lg
- 라운딩: rounded-md, rounded-lg

## 컴포넌트 작성 규칙
- 'use client' 디렉티브가 필요한 경우 명시 (useState, useEffect 사용 시)
- 서버 컴포넌트를 기본으로 사용
- 컴포넌트는 간결하고 재사용 가능하게 작성
- props 타입은 JSDoc으로 문서화
